# Yankees 2026 Schedule

New York Yankees 2026 regular season command center. Built with the VERITAS design system: gold, obsidian, live game state, roster context, broadcast data, and an official-radio-first listening flow.

## Features

- Complete 2026 Yankees schedule from the MLB Stats API
- Current-month schedule ordering with Today / Upcoming first
- Live score and pitch-by-pitch PitchCast panel during games
- Active Yankees roster grouped by position
- TV and radio broadcast badges on game cards and detail modals
- Radio Booth with in-app WFAN playback and official WFAN / WFAN2 / WADO fallback links
- Game detail modal with probable pitchers, broadcasts, MLB Gameday, tickets, and calendar export
- VERITAS Omega aesthetic
